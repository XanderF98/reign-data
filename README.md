# Seattle Reign: A Decade of Performance Insights (2016–2025)

## Overview
This project analyzes Seattle Reign FC’s performance in the NWSL from 2016–2025 (excluding 2020). It combines data visualization and storytelling to highlight trends, key milestones, and insights.

## Features
- **Interactive Dashboard**: Points vs Goals trend and scatter with regression.
- **Static Visuals**: PNG charts for quick reference.
- **Data File**: Clean CSV for reproducibility.

## Data Sources
- FBref for season standings, points, and goals.
- NWSL official site for playoff context.

## Visualizations
- **Trend Chart**: Points vs Goals For across seasons.
- **Scatter Plot**: Correlation between scoring and points.
- **Annotations**: Highlights for 2022 Shield win and 2024 dip.
